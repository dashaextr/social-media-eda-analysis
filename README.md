# Social Media Engagement Analysis (EDA)

## Overview
This project explores a social media dataset (~10,000 posts) to analyse user engagement patterns across platforms.

The analysis focuses on data quality, engagement distribution, platform differences, and temporal trends.

## Key Questions
- How is engagement distributed across posts?
- Are there differences between platforms?
- How does engagement change over time?
- What relationships exist between engagement metrics?

## Tools
- Python (pandas, matplotlib, seaborn)
- Exploratory Data Analysis (EDA)

## Key Insights
- Engagement metrics are highly right-skewed, with a small number of posts generating most interactions
- Significant missing values in views limit engagement rate analysis
- Engagement patterns vary across platforms
- No clear upward or downward trend over time — engagement fluctuates
- Likes, comments, and shares are positively correlated, but capture different aspects of user behaviour

## Visual Analysis
This project includes:
- Distribution plots (likes)
- Platform comparison (boxplots)
- Time-series analysis
- Heatmaps for platform and correlation analysis

## Business Implications
- Viral content drives the majority of engagement
- Platform-specific strategies are required
- Engagement metrics should be normalised for fair comparison
- Data quality improvements are essential for reliable analysis
